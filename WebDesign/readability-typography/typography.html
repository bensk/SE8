What's happening here?

There are three kinds of typefaces on this page:

1. Serif: look at a (printed) book. You'll find a serif typeface.

2. Sans serif: these look modern, technical, and lighter. Smartphones, the NYC Subway signs- all Sans serif.

3. Monospace: Ever seen a typewriter?

Deliberately using contrasting font types for different elements (body text vs. headings, for example) helps users more easily navigate content.

Coding connection: Why do you think we always see code written in monospace fonts?

I'm also deliberately using just 3 colors- one for body, one for headings, and one for code. Trying to follow the Principles of Flag Design. And I suppose one more for Flag...
